## Current Projects

### LLM Proxy

A lightweight Go-based reverse proxy for accessing multiple LLM providers through a unified gateway.

- **Multi-provider support**: OpenAI, Anthropic, Google Gemini
- **Unified API**: Single endpoint for all LLM access
- **Cost tracking**: Monitor and control LLM spending
- **Rate limiting**: Token-based and request-based limits
- **Streaming support**: Native handling of streamed responses
